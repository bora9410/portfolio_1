# Bora's Portfolio

## 프로젝트 목표
취업을 위한 포트폴리오

## 기술 스텍
- HTML
- CSS
- JavaScript
- jQuery
  
## 사용 프로그램
- Visual Studio Code
- SASS
- OpenWax (웹표준,접근성 진단)
- CCA (색 명암 분석기)
- W3C (마크업 검사)
- WAVE (접근성 검사기)

## 기능 및 섹션
- 자기소개 페이지
- 사용 가능한 기술 설명
- 포트폴리오 작품 리스트
- 연락처 정보 및 이력서 열람 페이지
